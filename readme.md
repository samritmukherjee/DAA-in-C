# Design and Analysis of Algorithms (DAA) Lab Programs

This repository contains the C implementations of the **Design and Analysis of Algorithms (DAA)** laboratory experiments as prescribed in the **MAKAUT (PCC-CS404)** syllabus.

Each algorithm is implemented in a separate source file with proper comments to help understand the logic and implementation.

---

## 📚 Repository Contents

### 1. Divide and Conquer
- Binary Search
- Merge Sort
- Quick Sort
- Maximum and Minimum Element (Divide & Conquer)

### 2. Dynamic Programming
- Matrix Chain Multiplication (MCM)
- Floyd–Warshall Algorithm (All-Pairs Shortest Path)

### 3. Graph Algorithms
- Dijkstra's Algorithm
- Bellman-Ford Algorithm
- Prim's Minimum Spanning Tree
- Kruskal's Minimum Spanning Tree
- Breadth First Search (BFS)
- Depth First Search (DFS)

### 4. Backtracking
- 8 Queen Problem
- Graph Coloring Problem
- Hamiltonian Cycle Problem

### 5. Greedy Algorithms
- Knapsack Problem
- Job Sequencing with Deadlines

### 6. Branch and Bound
- 15 Puzzle Problem

### 7. Other Algorithms
- Traveling Salesman Problem (TSP)
